AdvaitaHealth
===========================
## 项目介绍

用区块链可信任健康数据将商业从商品进化到服务，从服务到健康游戏竞赛，从健康游戏竞赛到共同参与数据收集的商业模式。在数据众包的基础上用人工智能与机器学习的方法探索适合人类的健康生活方式。

![Image text](https://raw.githubusercontent.com/towben/AdvaitaHealth/main/framework.png)


## 环境依赖

node v0.10.28+
reids ~

## 部署步骤
1. 安装:

```bash
yarn
```

2. 运行:

```bash
yarn test
```
